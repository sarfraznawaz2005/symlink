# SymLink Creator

## Project Overview
AutoHotkey v2 application for creating symbolic links via Windows context menu.

## Build & Test Commands

```bash
# Syntax check a single file (use AutoHotkey v2)
"C:\Program Files\AutoHotkey\v2\AutoHotkey.exe" /debug script.ahk

# Run main application
main.ahk

# Run context menu handler (requires path argument)
context_menu.ahk "C:\path\to\target"

# No traditional test runner - test by running scripts manually
```
